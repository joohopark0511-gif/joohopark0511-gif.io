# Advanced AI Chatbot

A simple AI chatbot that can answer questions and search Google using Node.js, Express, and Google Custom Search API.

## Features
- Chat interface (frontend in HTML/JS)
- Safety filter for sensitive topics
- Google search for queries
- Simple keyword-based AI fallback

## Requirements
- Node.js installed
- Google API Key & Custom Search Engine ID (CX)
